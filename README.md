# Quiz-Leaderboard-System

A backend-based Quiz Leaderboard System that manages quizzes, users, and rankings. This project provides REST APIs to handle quiz data, calculate scores, and display leaderboards.

---

## 🚀 Features

- 🧑 User Management (Create / View Users)
- ❓ Quiz Management
- 🧮 Score Calculation
- 🏆 Leaderboard Ranking
- 🔗 REST API Integration
- 📦 Structured Backend Architecture

---

## 🛠️ Tech Stack

- Java (Spring Boot)
- REST APIs
- Maven
- JSON
- Git & GitHub
